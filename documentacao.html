Frete — como funciona

Brasil (Mini Envios e outras modalidades)

  • Com Correios configurados (CORREIOS_USER + CORREIOS_PASSWORD + cartão ativo): cotação real via API dos Correios.
  • Sem credenciais ou contrato pendente: estimativa fixa por distância de CEP (R$ 11,90 a R$ 24,90). O checkout continua funcionando.
  • Modalidades ativas: Admin → aba Frete → “Modalidades de envio”.

Internacional

  • Cotação em tempo real: simulador Exporta Fácil dos Correios.
  • Se a API falhar: tabela fallback no admin (sincroniza automaticamente quando a API responde).

Status na aba API do admin

  • ✓ Conectado
  • ⚠ Aviso (sandbox, fallback, sem credenciais com estimativa)
  • ✗ Erro (credencial inválida ou serviço indisponível)
  • — Não configurado

Correios — passo a passo

  1. Cadastro Meu Correios com CNPJ da 3N20.
  2. Gerar código da API em CWS → Gestão de acesso a APIs.
  3. Contratar Clube Correios em Correios Empresas (aguardar até 2 dias úteis).
  4. Cartão de postagem: empresas.correios.com.br → Meus Contratos → Cartões de Postagem.
  5. Secrets: CORREIOS_USER (CNPJ), CORREIOS_PASSWORD (código CWS), CORREIOS_CONTRACT (10 dígitos).
  6. Validar: Admin → aba API → linha Correios BR deve ficar ✓.

Inscrição estadual (se pedirem no contrato): consulta pública CADESP. Se não houver IE ativa, use ISENTO.

Log de cliques (Admin → Cliques)

Registro automático de visitas e cliques no site. Serve para entender de onde vem o tráfego e o que as pessoas fazem — não substitui Pedidos (venda confirmada só aparece em /pedidos.html).

O que é “Acesso direto”?

O site não conseguiu identificar de onde a pessoa veio. Na prática, costuma ser:

  • Digitou sensortattoofix.com.br na barra do navegador
  • Favorito / bookmark
  • Link em WhatsApp, Telegram ou e-mail (muitos apps não enviam origem)
  • QR code ou cartão de visita
  • Bot, monitoramento ou servidor (também aparece assim)

Não dá para saber com certeza qual desses foi — só que não veio de anúncio rastreado nem de link com origem registrada.

Exemplo de jornada no log

Google Ads  →  Entrada — Home  →  Entrada — Loja  →  Entrada — Checkout

Interpretação: clicou no anúncio, navegou e chegou ao checkout. Ainda não é venda — confira em Pedidos se virou pedido pago.

Cores — duas colunas diferentes

  • Centro (verde): tipo de página ou clique (Entrada — Home, Mercado Livre…).
  • Direita (badge): origem do tráfego na primeira entrada da visita (Acesso direto, Google Ads…).

Nota técnica (jun/2026): em versões anteriores, “Acesso direto” podia aparecer em cinza ou amarelo para a mesma situação (bug ao salvar o referrer humanizado). Corrigido no admin — acesso direto fica sempre cinza; amarelo reservado para link de outro site.

Dicas

  • Passe o mouse sobre a linha para ver tooltip (campanha UTM, referrer, dispositivo).
  • Use Exportar Excel para análise por dia/semana/mês.
  • Testar log grava evento de diagnóstico — use Limpar testes depois.
  • Métricas agregadas também existem no Google Analytics 4.

Uber Direct (entregas rápidas BR)

Entrega sob demanda como opção extra no checkout (Brasil). Após pagamento confirmado, o Worker cria a corrida na API Uber e envia o link de rastreio por e-mail/WhatsApp.

Checkout: a cotação Uber só aparece com CEP + rua + cidade + UF preenchidos. O endereço de coleta usa shipping.sender e originCep do config.

Testes R$ 0,01: pedidos simbólicos (PIX mesmo endereço do remetente ou PayPal com PAYPAL_SELF_TEST) não criam corrida Uber, mesmo se o frete Uber for escolhido no checkout.
