Modalidades de envio (cliente escolhe)

O checkout lista preço e prazo de cada modalidade ativa. Internacional: tipo M = Exporta Fácil (encomenda), tipo D = documento/carta internacional (bem mais barato para lente fina). Use * para todos os serviços daquele tipo.

Pacote e frete (Brasil)

O checkout consulta a API Preço dos Correios para cada modalidade BR ativa quando CORREIOS_USER e CORREIOS_PASSWORD estão no Worker. Sem isso, usa estimativa fixa.
